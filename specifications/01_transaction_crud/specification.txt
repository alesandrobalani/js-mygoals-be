Como responsável pela família quero cadastrar os lançamentos financeiros ocorridos, categorizando estes lançamentos de forma a ter visibilidade das receitas e despesas ocorridas em um determinado período.

Para isto quero armazenar cada transação com as seguintes informações:
- Data da transação
- Tipo: Receita ou despesa
- Categoria: Habitação, Serviços públicos, Educação, Saúde, Alimentação, Transporte, Lazer, Cuidados pessoais, Renda Ativa, Renda extra, Renda passiva.
- Descrição
- Data de vencimento
- Conta

Este se trata de um projeto de backend, cuja a interface de comunicação será por REST.

Este projeto deve ser criado na linguagem JS.
